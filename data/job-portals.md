## How to Use This with career-ops

1. **Scan tracked companies:** `/career-ops scan` — hits Greenhouse/Ashby/Lever APIs + company career pages
2. **Manually search portals above:** Use queries like `"Java" "Spring Boot" "Backend Engineer" Dubai` on Bayt, Naukrigulf, LinkedIn
3. **Paste JD URL into pipeline:** Add interesting URLs to `data/pipeline.md`, then run `/career-ops pipeline`
4. **Evaluate before applying:** Run `/career-ops oferta` on any offer before spending time on it

**Tip:** For UAE portals (Bayt, Naukrigulf, GulfTalent), the scan script can't hit their APIs directly. Use the search queries manually or paste found URLs into `data/pipeline.md`.
